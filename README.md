# Qiita記事 検索アプリ

## Qiita API
- Qiita API v2
  - APIに関して（公式ドキュ）：
    - 利用制限認証している状態ではユーザーごとに1時間に1000回まで、認証していない状態ではIPアドレスごとに1時間に60回までリクエストを受け付けます。
  - https://qiita.com/api/v2/docs
